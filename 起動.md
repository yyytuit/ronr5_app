- 起動

docker-compose up

- シャットダウン

docker-compose down

- rails コンソール等

docker-compose run web rails c

- サンプルデータの挿入

docker-compose run web rails db:fixtures:load

- 指定の位置までマイグレーションを戻す

docker-compose run web rails db:rollback STEP=4

- migration のステータス

docker-compose run web rails db:migrate:status
